# 인공지능사관학교 7기 미니프로젝트 1주차
헬스케어 - AI 운동 코치

## 멤버
- 김민호 : PM
- 박종효 : UI/UX
- 선범수 : AI/프론트엔드
- 하지원 : 백엔드

## Pain points
- 피드백 부재
  - 혼자 운동할 때 내 자세가 맞는지 확신이 안 선다
  - 유튜브 따라 하는데 거울로 봐도 제대로 하는지 모르겠다
  - 영상으로 찍어봐도 뭐가 틀렸는지 내 눈엔 안 보인다
  - 맨몸운동의 정확한 가동범위(ROM)를 모른다
- 비용·접근성
  - PT는 비싸서 매번 받을 수 없다 (회당 5~7만 원)
  - 헬스장 트레이너에게 매번 봐달라고 하기 눈치 보인다
- 부상·불안
  - 스쿼트할 때 무릎이 아픈데 자세 문제인지 모르겠다
  - 데드리프트 후 허리가 아픈데 원인을 모른다
  - 부상 경험이 있어 무게 올리는 게 무섭다
  - 재활운동을 처방받았는데 집에서 맞게 하는지 불안하다
- 효과·동기
  - 같은 운동을 반복해도 어느 근육에 들어가는지 모른다
  - 운동 효과가 안 나는데 자세 탓인지 강도 탓인지 모른다
  - 맨몸운동의 정확한 가동범위(ROM)를 모른다

사용자가 운동을 그만두는 진짜 이유는 게을러서가 아니라 "내가 맞게 하는지 몰라서 불안하기" 때문이다. 이 감정 곡선이 문제 정의의 핵심 근거

## Persona

### 메인 페르소나 — 김도현 (30세) · 근력·체형 관리
- 기본 정보
  - 30세 / IT 회사 백엔드 개발자
  - 서울 강남구 · 1인 가구
  - 회사 헬스장 + 홈트, 주 3~4회, 거의 항상 혼자
- 건강 상태
  - 큰 부상 이력 없음, 거북목 + 가벼운 허리 뻐근함
  - 종일 앉아 일 → 운동으로 만회하려 함
- 페인포인트
  - 내 자세가 맞는지 확신이 없다
  - 다칠까 봐 겁나서 늘 가벼운 무게만 든다
  - PT는 비용·시간 부담이라 안 받는다
- 목표 (Need)
  - 부상 없이, 제대로 운동하고 싶다
  - 확신을 갖고 무게를 점진적으로 올리고 싶다
  - 비싼 PT 없이 객관적 피드백을 받고 싶다
- 기술 친화도
  - 높음 — 앱·웨어러블·유튜브에 익숙, 카메라 분석에 거부감 없음

### 서브 페르소나 ① — 박동희 (33세) · 시간 빈곤·체중 관리
- 기본 정보
  - 33세 / 영업직
  - 서울 목동 · 기혼 (아내, 자녀 1)
- 건강 상태
  - 과체중 (관리 요망)
  - 복용 약 없음
- 페인포인트
  - 육아·업무로 운동할 시간이 절대적으로 부족
  - PT 비용이 부담된다
  - 혼자 하는 운동에 회의감 — '잘하고 있는지 모르겠다'
- 목표 (Need)
  - 올바른 자세로 부상 없이 꾸준히 운동해 체중 관리의 토대를 만들고 싶다 (장기적으로 체중 10kg 감량)
  - 타인의 시선 없이 나만의 페이스로, 전문적 피드백을 받아 올바른 방향으로 운동하고 싶다
- 기술 친화도
  - 높음 — 스마트폰 익숙, 짧은 시간에 빠르게 쓸 수 있는 도구 선호 (시간 빈곤형)

### 서브 페르소나 ② — 이지은 (38세) · 재활·통증 관리
- 기본 정보
  - 38세 / 재택 프리랜서 디자이너
  - 경기 성남시 · 배우자·자녀와 거주
  - 홈트 중심 (헬스장 갈 시간이 없음)
- 건강 상태
  - 출산 후 코어 약화 + 간헐적 허리 통증
  - 정형외과에서 재활운동 권유받음
- 페인포인트
  - 재활운동을 맞게 하는지 불안해 결국 안 하게 된다
  - 잘못하면 통증이 재발할까 두렵다
  - 헬스장·필라테스 갈 시간이 없다
- 목표 (Need)
  - 안전하게 통증을 관리하고 코어를 회복하고 싶다
  - 집에서 정확한 자세로 재활운동을 하고 싶
  - '지금 잘하고 있다'는 안심을 받고 싶다
- 기술 친화도
  - 중간 — 복잡한 건 부담, 큰 글씨·단순 화면·명확한 OK/교정 표시 선호
